AIJ Video Viewer

Простое веб‑приложение на React + TypeScript + Vite для просмотра видео по публичным ссылкам Яндекс.Диска. Преобразует ссылку вида `https://disk.yandex.ru/d/<ключ>/<путь к файлу>` в прямую ссылку через публичный API Яндекс.Диска и воспроизводит видео через тег `<video>`.

Запуск

- Установите зависимости: `npm i`
- Запустите dev‑сервер: `npm run dev`
- Откройте адрес из терминала (обычно `http://localhost:5173`)

Использование

- Вставьте публичную ссылку вида `https://disk.yandex.ru/d/.../file.mov` или прямую ссылку на `.mp4/.webm/.mov`.
- Нажмите «Загрузить». Приложение получит прямую ссылку через `https://cloud-api.yandex.net/v1/disk/public/resources/download` и начнет воспроизведение.
- Текущая исходная ссылка сохраняется в `localStorage`, поэтому после перезагрузки страницы последнее введённое значение подставляется автоматически.

Примечания

- Ссылки должны быть публичными. Авторизация не требуется.
- Поддерживаются домены `disk.yandex.ru` и `disk.*.yandex.ru` (например, `disk.360.yandex.ru`) — любые такие ссылки резолвятся через публичный API.
- Если видео не воспроизводится, проверьте, что ссылка действительно указывает на файл видео, а не на папку.
- Некоторые файлы `.mov` могут использовать кодеки, не поддерживаемые вашим браузером. В таком случае попробуйте другой браузер или перекодируйте видео.
